# Terrain Map (.T2)

FA_2.LIB contains 16 `.T2` files. Each stores the height, color, and surface-type grid for one theater's terrain — the tile map the engine uses to render ground elevation and textures during flight. Referenced by name from `.MM` theater files via the `map` keyword.

## Format

Binary with a fixed ASCII magic and name header.

```
Offset  Size  Description
------  ----  -----------
0x00    4     Magic: "BIT2" (ASCII, no null)
0x04    60    Theater name, null-padded ASCII (e.g. "Panama\0\0...")
0x40    ...   Terrain data (structure unknown)
```

`APA.T2` (theater "Panama") is **199,829 bytes** — the largest format encountered in FA_2.LIB, consistent with a full 2D terrain grid.
